# Prognostic-Cell-Type-Classification-for-Acute-Myeloid-Leukemia
Prognostic Cell Type Classification for AML with Deconvolution and Non negative Matrix factorisation
Dataset used TCGA LAML bulk rna data and single cell rna data from __ blood.h5ad and bonemarrow.h5ad
run python scripts in the scripts/ folder to build the deconvolution model and obtain figures
